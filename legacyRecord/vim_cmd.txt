//-------------------------------vim编辑器的使用-------------------

光标移动：

使用 h、j、k、l 键来左、下、上、右移动光标
使用 ^ 键将光标移至行首
使用 $ 键将光标移至行尾
使用 gg 键将光标移至文档开头
使用 G 键将光标移至文档结尾
使用 w 键将光标向后移动一个单词
使用 b 键将光标向前移动一个单词
使用 Ctrl + u 键向上滚动半个屏幕
使用 Ctrl + d 键向下滚动半个屏幕

文本选择：

使用 v 键进入可视模式，使用箭头键选择文本
使用 V 键进入行选择模式，使用箭头键选择行
使用 Ctrl + v 键进入块选择模式，使用箭头键选择块
	从光标当前位置开始，选中光标起点和终点所构成的矩形区域，再按一下Ｃtrl + v结束。 

复制、剪切与粘贴：

使用 y 键复制选中的文本
使用 d 键剪切选中的文本
使用 p 键在光标后粘贴文本
使用 P 键在光标前粘贴文本

删除与撤销：

使用 x 键删除光标下的一个字符
使用 dd 键删除光标所在行
使用 u 键撤销上一步操作
查找和替换：
使用 / 字符和关键词来查找文本，使用 n 和 N 键移动到下一个和上一个匹配项
使用 :%s/old/new/g 命令来替换文本，其中 old 是要被替换的字符串，new 是替换后的字符串，g 表示全局替换

用vim时，想高亮显示一个单词并查找的方发，将光标移动到所找单词.

1: shift + "*"  向下查找并高亮显示

2: shift + "#"  向上查找并高亮显示

3: "g" + "d"    高亮显示光标所属单词，"n" 查找！
