Ctrl + Alt + T：打开终端窗口
Ctrl + Alt + F1~F6：切换到虚拟终端
Ctrl + Alt + F7：切换回图形界面
Alt + Tab：切换窗口
Alt + F4：关闭当前窗口
Ctrl + C：终止当前进程
Ctrl + A：将光标移至行首
Ctrl + E：将光标移至行尾
Ctrl + Z：将当前进程置于后台
Ctrl + Shift + T：打开新的标签页
Ctrl + Shift + W：关闭当前标签页
Ctrl + Shift + N：打开新的文件管理器窗口
Ctrl + Q：退出当前应用程序
Ctrl + W：关闭当前标签页或窗口
Alt + F2：打开运行命令对话框

光标移动：
使用箭头键（上、下、左、右）来移动光标
使用 Home 键将光标移至行首
使用 End 键将光标移至行尾
使用 PageUp 键将光标向上移动一页
使用 PageDown 键将光标向下移动一页
文本选择：
使用 Shift 键加上箭头键来选择文本
使用 Ctrl + A 来全选文本
复制、剪切与粘贴：
使用 Ctrl + C 来复制选中的文本
使用 Ctrl + X 来剪切选中的文本
使用 Ctrl + V 来粘贴文本
删除与撤销：
使用 Backspace 键来删除光标前的一个字符
使用 Delete 键来删除光标后的一个字符
使用 Ctrl + Z 来撤销上一步操作
窗口控制：
使用 Alt + Tab 来切换窗口
使用 Alt + F4 来关闭当前窗口
使用 Ctrl + Alt + T 来打开终端窗口
打开文件属性：
使用 Alt + Enter 快捷键可以打开所选文件的属性对话框。

窗口管理
Alt + F7 键：使用这个组合键，您可以移动当前窗口。

按下这个组合键后，您可以使用箭头键或鼠标来移动窗口位置。
如果您使用的是桌面环境中的 Openbox 窗口管理器，则可以使用这个组合键来移动所有窗口。

Alt + F8 键：使用这个组合键，您可以调整当前窗口的大小。
按下这个组合键后，您可以使用箭头键或鼠标来改变窗口大小。
如果您使用的是桌面环境中的 Openbox 窗口管理器，则可以使用这个组合键来改变所有窗口的大小。

Alt + F9 键：使用这个组合键，您可以最小化当前窗口。
按下这个组合键后，当前窗口会最小化到任务栏。

Alt + F10 键：使用这个组合键，您可以最大化或还原当前窗口。
按下这个组合键后，当前窗口会最大化或还原到原始大小。
